## Installation


Install the dependencies and devDependencies

[Node.js](https://nodejs.org/)

```sh
cd /backend
npm install
npm start
```

[React.js](https://reactjs.org/)

```sh
cd /frontend
npm install
npm start
```

# About

chat project using Socket.io, the client enters a name + a room to choose
